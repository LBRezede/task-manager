# Task Manager

Projeto independente de gerenciamento de tarefas com Node.js, React e SQL.

## Funcionalidades

- Cadastro e login de usuários
- CRUD de tarefas
- Interface responsiva e moderna

## Tecnologias

- Backend: Node.js com Express
- Banco de Dados: PostgreSQL ou MySQL
- Frontend: React
- Autenticação: JWT
